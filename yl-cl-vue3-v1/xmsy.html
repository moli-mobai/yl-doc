a、proxy ​

composition api 中的 setup 函数的执行并没有绑定实例对象，所以 vue3 的 setup 中无法使用 this，于是定义proxy代替 this。

js
# vue2使用消息弹窗
this.$modal.msgSuccess('删除成功');

# vue3使用消息弹窗（proxy代替this）
import { getCurrentInstance, ComponentInternalInstance } from 'vue';
const { proxy } = getCurrentInstance() as ComponentInternalInstance;
proxy!.$modal.msgSuccess('删除成功');

温馨提示！

因为是 ts 语法，若只写proxy.$modal，代码规范 elisnt 检测会报错。

所以proxy需要搭配!或者?，proxy!用于自定义方法(必定存在)，proxy?若不存在，则不再往下运算，用于不确定对象、方法。

b、最小化弹窗 ​

描述的比较麻烦，看不懂的话找我实际操作一下明白了！

重点: 一共 3 模块，父组件、desk 组件(中间件-事件处理)、子组件(弹窗页)。

desk 中间件已封装完毕，使用者一共需要配置 2 步，弹窗 组件化(子组件) ，父组件 具体配置事项请看下方。

1.第一步 子组件的配置，拆分步骤教学。 ​

(2)、弹窗还原事件 getOpen()

html
# defineExpose暴露给父组件调用(不理解看vue3文档)
<script setup lang="ts">
  function getOpen() {
    open.value = true;
  }

(3)、弹窗关闭事件，父组件监听调用 emit('toDelete',onlyKey)

html
# 注意事项 onlyKey 和 emit
<script setup lang="ts">
  const onlyKey = ref("");
  function init(only: any = "") {
    onlyKey.value = only;
  }

  const emit = defineEmits(["toDelete"]);
  function cancel() {
    emit("toDelete", onlyKey.value);
  }

  defineExpose({ init });
</script>

解释说明

  1. onlyKey 是组件唯一 key，init 初始化时接收，关闭弹窗时返回父组件。

  2. 需要关闭弹窗时触发 emit("toDelete", onlyKey)，如 cancel 事件等。

  3. defineEmits 跟 defineExpose 有什么区别？？？

    defineEmits 是子组件告诉父组件，你快启动自定义方法。

    defineExpose 是父组件主动调用子组件方法。

    总结: 一个是子组件主动，一个是父组件主动，defineEmits 就是 vue2 里的 emit

defineExpose 方法参数、属性如何使用解释说明

js
1. itemRefs 弹窗组件列表数据。
let arr = deskRef.value?.itemRefs;

2. isIndexId(:onlyKey) 根据id查询弹窗组件索引
# onlyKey为唯一值，不可重复
let isHas = deskRef.value?.isIndexId(onlyKey);

3. goOpen(isHas) 根据上方索引打开弹窗组件
deskRef.value?.goOpen(isHas)

4. 组件索引 isHas不存在？getObj创建组件信息
import { shallowRef } from 'vue';
import { xxxDialog } from 'xxxDialog';

if(isHas) return deskRef.value?.goOpen(isHas)
let obj = deskRef.value?.getObj(shallowRef(xxxDialog), onlyKey, title);

5. 根据新创建的组件信息obj，生成组件并打开。
nextTick(() => {
  let i: any = deskRef.value?.isIndexId(onlyKey);
  deskRef.value?.itemRefs[i]?.init(onlyKey, title ,...); // 生成打开

});

6. 关闭组件
# toDelete 第一个参数为onlyKey，第二个参数为随意字符串或数字。
deskRef.value?.toDelete(onlyKey, string);
